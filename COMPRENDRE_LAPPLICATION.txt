🎯 C'EST QUOI ?

Une application qui AUTOMATISE vos tâches répétitives.

Vous définissez QUOI faire et QUAND le faire.
L'application fait le reste, 24/7, automatiquement !

═══════════════════════════════════════════════════════════════
    🔍 QU'EST-CE QU'ELLE PEUT FAIRE ?
═══════════════════════════════════════════════════════════════

1️⃣ TRAITER DES FICHIERS EXCEL
   • Lire des fichiers Excel
   • Extraire des données
   • Faire des calculs
   • Générer des rapports

   Exemple : "Tous les matins à 8h, lis le fichier des ventes
             et génère un rapport avec graphiques"

2️⃣ ENVOYER/RECEVOIR DES EMAILS
   • Lire les emails entrants
   • Télécharger les pièces jointes
   • Envoyer des emails automatiquement
   • Répondre automatiquement

   Exemple : "Quand un client envoie une commande par email,
             traite-la et renvoie une confirmation"

3️⃣ SE CONNECTER À DES APIS
   • Récupérer des données d'une API externe
   • Envoyer des données
   • Synchroniser automatiquement

   Exemple : "Toutes les heures, synchronise les données
             avec l'API CORTEX"

4️⃣ ENVOYER DES NOTIFICATIONS
   • Email
   • WhatsApp
   • Amazon Chime

   Exemple : "Quand le traitement est terminé, envoie un
             message WhatsApp au responsable"

5️⃣ PLANIFIER L'EXÉCUTION
   • Tous les X minutes
   • Toutes les X heures
   • Tous les jours à une heure précise
   • Chaque semaine (lundi, mardi...)
   • Chaque mois (1er, 15...)

   Exemple : "Tous les lundis à 9h" ou "Toutes les 5 minutes"

═══════════════════════════════════════════════════════════════
    🎓 LES CONCEPTS CLÉS
═══════════════════════════════════════════════════════════════

TÂCHE
─────────────────────────────────────────────────────────────
= Une action automatisée que vous définissez

Exemples :
• "Générer un rapport"
• "Traiter les emails"
• "Synchroniser avec l'API"

PLANIFICATION
─────────────────────────────────────────────────────────────
= QUAND la tâche doit s'exécuter

Exemples :
• "Tous les jours à 8h"
• "Toutes les 5 minutes"
• "Chaque lundi"

JOB / EXÉCUTION
─────────────────────────────────────────────────────────────
= Une instance d'exécution d'une tâche

Exemple :
• Job #1234 : "Rapport quotidien" exécuté le 21/10 à 08:00

LOG
─────────────────────────────────────────────────────────────
= L'enregistrement de ce qui s'est passé

Exemple :
• ✅ Succès
• Durée : 5 secondes
• Fichier créé : rapport.xlsx

MODULE
─────────────────────────────────────────────────────────────
= Un composant de l'application avec une fonction spécifique

Exemples :
• Module Excel : traite les fichiers Excel
• Module Email : gère les emails
• Module API : se connecte aux APIs
