═══════════════════════════════════════════════════════════════
    🎬 DÉMONSTRATION RAPIDE - NOUVEAU SYSTÈME
═══════════════════════════════════════════════════════════════

⏱️ Durée : 2 minutes

═══════════════════════════════════════════════════════════════
    ÉTAPE 1 : CONNEXION (10 secondes)
═══════════════════════════════════════════════════════════════

1. Ouvrir : http://localhost:4300
2. Email : admin@example.com
3. Mot de passe : admin123
4. Cliquer sur "Se connecter"

═══════════════════════════════════════════════════════════════
    ÉTAPE 2 : CRÉER UNE TÂCHE QUOTIDIENNE (30 secondes)
═══════════════════════════════════════════════════════════════

1. Cliquer sur "Tâches" dans le menu de gauche
2. Cliquer sur "+ Nouvelle tâche" (bouton bleu en haut à droite)
3. Remplir :
   
   Nom : Rapport matinal
   Type : Routenverfuegbarkeit
   Description : Génération du rapport quotidien
   
4. Section Planification :
   
   Fréquence : [Choisir "Quotidien"]
   À quelle heure ? : 08:00
   
   📘 Un résumé s'affiche : "Tous les jours à 08:00"
   
5. ✓ Cocher "Activée"
6. Cliquer sur "Créer"

✅ Tâche créée ! Elle apparaît dans la liste.

═══════════════════════════════════════════════════════════════
    ÉTAPE 3 : CRÉER UNE TÂCHE HEBDOMADAIRE (30 secondes)
═══════════════════════════════════════════════════════════════

1. Cliquer à nouveau sur "+ Nouvelle tâche"
2. Remplir :
   
   Nom : Rapport hebdomadaire
   Type : Staging Plan
   Description : Bilan de la semaine
   
3. Section Planification :
   
   Fréquence : [Choisir "Hebdomadaire"]
   Quel jour ? : Lundi
   À quelle heure ? : 09:00
   
   📘 Résumé : "Chaque lundi à 09:00"
   
4. ✓ Cocher "Activée"
5. Cliquer sur "Créer"

✅ Deuxième tâche créée !

═══════════════════════════════════════════════════════════════
    ÉTAPE 4 : TESTER UNE TÂCHE (20 secondes)
═══════════════════════════════════════════════════════════════

1. Dans la liste des tâches
2. Trouver "Rapport matinal"
3. Cliquer sur le bouton vert "▶ Exécuter"
4. Confirmer l'exécution

📝 Message : "Tâche lancée. Job ID: xxx"

═══════════════════════════════════════════════════════════════
    ÉTAPE 5 : VOIR LES LOGS (15 secondes)
═══════════════════════════════════════════════════════════════

1. Cliquer sur "Logs" dans le menu de gauche
2. Voir l'exécution qui vient d'être lancée
3. Statut de la tâche :
   ✅ Succès (vert)
   ❌ Échec (rouge)
   ⏱️ En cours (jaune)

═══════════════════════════════════════════════════════════════
    ÉTAPE 6 : DASHBOARD (15 secondes)
═══════════════════════════════════════════════════════════════

1. Cliquer sur "Dashboard" dans le menu
2. Voir les statistiques :
   • Nombre de tâches actives
   • Exécutions récentes
   • Performance par tâche
   • Graphiques

═══════════════════════════════════════════════════════════════
    🎉 FÉLICITATIONS !

✅ Exécuter une tâche manuellement

✅ Voir les statistiques

🔄 ACTIVER / DÉSACTIVER UNE TÂCHE
   → Toggle ON/OFF dans la liste des tâches
   
✏️ MODIFIER UNE TÂCHE
   → Cliquer sur le bouton bleu "✏"
   → Changer la planification
   → Sauvegarder
   
🗑️ SUPPRIMER UNE TÂCHE
   → Cliquer sur le bouton rouge "🗑"
   → Confirmer la suppression
   
📋 AUTRES FRÉQUENCES
   → "Toutes les X minutes" : Pour les tâches fréquentes
   → "Toutes les X heures" : Pour les vérifications régulières
   → "Mensuel" : Pour les rapports de fin de mois
