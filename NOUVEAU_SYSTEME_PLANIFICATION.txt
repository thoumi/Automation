═══════════════════════════════════════════════════════════════
    📋 LES 5 OPTIONS DE PLANIFICATION
═══════════════════════════════════════════════════════════════

1️⃣ TOUTES LES X MINUTES
   → Pour des tâches fréquentes
   → Exemples : Toutes les 5 minutes, Toutes les 15 minutes
   
2️⃣ TOUTES LES X HEURES
   → Pour des tâches plusieurs fois par jour
   → Exemples : Toutes les heures, Toutes les 3 heures
   
3️⃣ QUOTIDIEN
   → Une fois par jour à une heure précise
   → Exemple : Tous les jours à 09:00
   
4️⃣ HEBDOMADAIRE
   → Une fois par semaine, un jour spécifique
   → Exemple : Chaque lundi à 08:30
   
5️⃣ MENSUEL
   → Une fois par mois, un jour spécifique
   → Exemple : Le 1er de chaque mois à 07:00

═══════════════════════════════════════════════════════════════
    🚀 COMMENT CRÉER UNE TÂCHE
═══════════════════════════════════════════════════════════════

ÉTAPE 1 : Aller sur http://localhost:4300
ÉTAPE 2 : Cliquer sur "Tâches" dans le menu
ÉTAPE 3 : Cliquer sur "Nouvelle tâche"
ÉTAPE 4 : Remplir :
   • Nom de la tâche
   • Type de tâche (Routenverfügbarkeit, Staging Plan, DNR Units)
   • Description (optionnelle)
   
ÉTAPE 5 : Dans la section "Planification" :
   • Choisir la fréquence dans la liste déroulante
   • Remplir les champs selon votre choix :
     - Minutes/Heures : Intervalle
     - Quotidien : Heure (ex: 09:00)
     - Hebdomadaire : Jour + Heure
     - Mensuel : Jour du mois + Heure
   
ÉTAPE 6 : Un RÉSUMÉ s'affiche automatiquement en bleu :
   "Tous les jours à 09:00"
   "Chaque lundi à 08:30"
   "Toutes les 5 minutes"
   
ÉTAPE 7 : Cocher "Activée" pour que la tâche démarre
ÉTAPE 8 : Cliquer sur "Créer"

C'EST TOUT ! 🎉

═══════════════════════════════════════════════════════════════
    🎮 GESTION DES TÂCHES
═══════════════════════════════════════════════════════════════

▶️  EXÉCUTER MAINTENANT
    → Bouton vert "▶" dans la liste
    → Test immédiat de la tâche
    
⏸️  ACTIVER / DÉSACTIVER
    → Toggle ON/OFF dans la liste
    → Pause temporaire sans supprimer la tâche
    
✏️  MODIFIER
    → Bouton bleu "✏" dans la liste
    → Changer la planification à tout moment
    
🗑️  SUPPRIMER
    → Bouton rouge "🗑" dans la liste
    → Suppression définitive

═══════════════════════════════════════════════════════════════
    📊 SUIVI DES EXÉCUTIONS
═══════════════════════════════════════════════════════════════

Dans la page "LOGS", vous voyez :

✅ Succès (en vert)
   → Tâche exécutée correctement
   → Durée d'exécution affichée
   
❌ Échec (en rouge)
   → Message d'erreur détaillé
   → Permet de corriger le problème
   
⏱️ En cours (en jaune)
   → Tâche en cours d'exécution

═══════════════════════════════════════════════════════════════
    🆘 CONSEILS PRATIQUES
═══════════════════════════════════════════════════════════════

💡 TESTER AVANT D'ACTIVER
   1. Créer la tâche
   2. Cliquer sur "▶ Exécuter"
   3. Vérifier les logs
   4. Si OK → Activer la planification
   
⏰ CHOISIR LE BON MOMENT
   • Rapports quotidiens → Tôt le matin (6h-8h)
   • Tâches lourdes → La nuit (2h-5h)
   • Vérifications → Toutes les heures pendant la journée
   
🔋 OPTIMISER LES PERFORMANCES
   • Ne pas lancer toutes les tâches en même temps
   • Espacer les horaires si plusieurs tâches quotidiennes
   • Désactiver les tâches inutiles temporairement
