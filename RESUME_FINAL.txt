═══════════════════════════════════════════════════════════════
    🎨 LES 5 OPTIONS DISPONIBLES
═══════════════════════════════════════════════════════════════

1. 📍 TOUTES LES X MINUTES
   Interface :
   - Intervalle : 1-60 minutes
   Résumé : "Toutes les 5 minutes"
   
2. 🕐 TOUTES LES X HEURES
   Interface :
   - Intervalle : 1-24 heures
   Résumé : "Toutes les 2 heures"
   
3. 📆 QUOTIDIEN
   Interface :
   - Heure : HH:mm
   Résumé : "Tous les jours à 09:00"
   
4. 📅 HEBDOMADAIRE
   Interface :
   - Jour : Lundi-Dimanche
   - Heure : HH:mm
   Résumé : "Chaque lundi à 08:30"
   
5. 📊 MENSUEL
   Interface :
   - Jour du mois : 1-31
   - Heure : HH:mm
   Résumé : "Le 1 de chaque mois à 07:00"

═══════════════════════════════════════════════════════════════
    💡 EXEMPLES DE CONVERSION
═══════════════════════════════════════════════════════════════

Interface Simple         →  Expression CRON  →  Description
──────────────────────────────────────────────────────────────
Quotidien à 09:00        →  0 9 * * *        →  "Tous les jours à 09:00"
Toutes les 5 minutes     →  */5 * * * *      →  "Toutes les 5 minutes"
Lundi à 08:30            →  30 8 * * 1       →  "Chaque lundi à 08:30"
Le 1er à 06:00           →  0 6 1 * *        →  "Le 1 de chaque mois..."
Toutes les 2 heures      →  0 */2 * * *      →  "Toutes les 2 heures"

LA CONVERSION EST 100% AUTOMATIQUE ! 🤖
